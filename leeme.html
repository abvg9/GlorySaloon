Comentarios al profesor

La web solo necesita como datos inciales un usuario de tipo admin. Si este usuario es borrado y no existe otro con permisos de administrador, parte de la funcionalidad no se podrá probrar. Solo un administrador puede cear otras cuentas que tengan privilegios de admin y todos pueden crear cuentas sin privlegios, incluso sin estar registrados. El usuario que hemos insertado se llama : manuel y su contraseña es aa. Hemos decidido no aportar datos de carga iniciales ya que con este usuario se pueden probar todas las funcionalidades desde cero.

Vistas

Detalla aquí tus vistas. Usa enlaces a localhost:8080/ (SIN nombre de proyecto; correspondientes a arrancar la aplicación en modo spring-boot desde STS) para enlazar cada vista.
  • vista principal - descripción muy corta de la vista
  • foro - Foro de debate de los miembros de la web
  • home - Pagina de bienvenida a la web
  • login - Pantalla donde los usuario entran a la web
  • partidaBlackJack - Lugar donde se desarrolla la partida de black jack
  • perfil - Información del usuario
  • perfilAmigo - Información de los amigos del usuario
  • ranking - Ranking en funcion del dinero de los usuarios
  • reglas - Pagina explicatoria de las reglas de los juegos implementados
  • saloon - Punto de encuentro entre los jugadores para unirse a una partida
  • tienda - Lugar donde los usuarios pueden compar items
